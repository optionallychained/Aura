Class ShaderProgram

Class representing a complete ShaderProgram

Stores and provides access to all the information the WebGLRenderer, EntityshaderMap and EntityManager need to render an Entity with a given shader pair

Used in registering shader programs on the game/renderer and for associating with Entities via Shader components
