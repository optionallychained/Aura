Interface StateConfig<TGame>

Interface describing a State Configuration object, providing a name as well as its lifecycle methods

The typeparam specifies which concrete Game type the State belongs to, allowing for the concrete State2D and State3D to receive a type-correct Game instance in their lifecycle methods, and enabling the assurance that a Game is only configured with the corresponding 2D or 3D State type

Type parameters

  • TGame: Game2D | Game3D

    the concrete Game Type the State belongs to
